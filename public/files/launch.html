HTML (HyperText Markup Language)

Como decíamos en la introducción, la información en una web app se muestra al usuario a través del HTML. En el HTML vamos a definir qué información se tiene que mostrar.

HTML vs CSS

HTML ➡️ QUÉ se muestra

CSS ➡️ CÓMO se muestra

HTML es un lenguaje de marcas, nos ofrece ciertas marcas (más conocidas como etiquetas) para estructurar la información. Por ejemplo, para crear un párrafo usamos las marcas <p>...</p>, para crear un título usamos la marca <h1>...</h1>, y así sucesivamente.
